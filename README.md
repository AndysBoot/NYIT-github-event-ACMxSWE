🧠 What is Git & GitHub?
Git is a version control system that helps you track changes in your code.
GitHub is a platform that hosts Git repositories and allows collaboration.

Think of Git as the tool, and GitHub as the place where your projects live online.

⚙️ Getting Started

Before you begin, make sure you have:

A GitHub account
Git installed on your computer
👉 Download Git: https://git-scm.com/downloads

Check installation:

git --version
📥 Cloning This Repository

To get a copy of this repository on your local machine:

git clone https://github.com/AndysBoot/NYIT-github-event-ACM.git

Then navigate into the folder:

cd NYIT-github-event-ACM
📁 Projects

Inside this repository, you’ll find a folder called:

/projects

This folder contains multiple beginner-friendly projects. Each project is designed for you to:

Explore the code
Modify it
Experiment freely

💡 Tip: You can copy any project and use it as the base for your own repository!

✏️ Making Your Own Changes
Open the project folder you want to work on
Edit files using any code editor (VS Code recommended)
Add new features, change styles, or break things (and fix them!)
💾 Saving Your Work (Git Workflow)
Step 1: Check Status
git status
Step 2: Add Changes
git add 
Step 3: Commit Changes
git commit -m "Describe what you changed"
☁️ Pushing Your Work

To upload your changes to GitHub:

git push origin main

If you're working in your own repository, make sure it’s connected properly.

🔑 Basic Git Commands
Command	Description
git clone	Copy a repository to your local machine
git status	Show changes in your project
git add .	Stage all changes
git commit -m "message"	Save changes with a message
git push	Upload changes to GitHub
git pull	Get latest changes from GitHub
